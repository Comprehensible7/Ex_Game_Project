# Python Game Project
PythonのLibraryであるPygameを利用して簡単なゲームを制作。

## 目的

基本的なCoding意外に、Libraryを利用しながら簡単なゲームを制作して技術的に能力を向上させたいと思いました。

## 開発環境

- Python 3.9
- Pygame 2.0
- VScode for Mac

## プロジェクト要求事項

- Pythonの知識が必要です。
- 基本的にPygameのSettingが必要です。
- Play Timeを確認できます。
- ボールに当たったらゲームが終了します。
- ボールをすべて取り除けば、'Mission Complete'が確認できます。
- 方向キーを操作するとキャラクターを動かすことができ、武器を発射できます。
